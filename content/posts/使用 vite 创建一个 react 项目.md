### 使用 Vite 指定 react 模版创建项目
```bash
  pnpm create vite personal-toolbox --template react
  cd personal-toolbox
  pnpm install
  pnpm run dev

```
### 了解 Vite
Vite 是一种新型前端构建工具，可提升前端开发体验。
